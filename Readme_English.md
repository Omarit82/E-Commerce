# E-Commerce

Example of an E-commerce application developed for the Backend course at Coderhouse.

## Technologies Used

- Developed with Node.js

- Express – Express Session – Express Handlebars

- JWT for authentication

- Passport with GitHub and local strategies

- MongoDB

- SweetAlert for the frontend with Handlebars

## Features

- Full CRUD

- Inventory and stock management

- Exception handling

- Data validation

## Setup and Installation

# Clone the repository:

git clone https://github.com/Omarit82/E-Commerce

# Navigate to the project directory and run:

npm install

# Finally, run:

npm run start


The project will be available at: http://localhost:8080

### The project includes a .env.dev file where you can see the names of the environment variables that must be completed in order to establish a successful connection.
